# Componentes de administración (`/components/admin`)
Esta carpeta contiene los componentes específicos del panel de administración. Están diseñados
para ser utilizados exclusivamente por usuarios autenticados con permisos de revisión.

---

## 📁 Componentes disponibles

| Componente      | Propósito                                                                 |
|------------------|--------------------------------------------------------------------------|
| `AdminPanel.jsx` | Muestra las sugerencias pendientes y permite aprobar o rechazar cada una |
