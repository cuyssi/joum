# Componentes comunes (`/components/common`)

Esta carpeta contiene componentes reutilizables y genéricos que se utilizan en distintas partes
de la aplicación. Están diseñados para ser simples, flexibles y fácilmente combinables.

---

## 📁 Componentes disponibles

| Componente                         | Propósito                                                                  |
|------------------------------------|----------------------------------------------------------------------------|
| `Button.jsx`                       | Botón reutilizable con texto, evento de clic y clases personalizadas       |
| `Footer.jsx`                       | Pie de página con créditos y enlaces a redes sociales                      |
| `Form.jsx`                         | Contenedor del flujo de traducción: resultado, pronunciación y reproductor |
| `Header.jsx`                       | Cabecera principal con el logotipo de Joum                                 |
| `Modal.jsx`                        | Ventana modal reutilizable con cierre por fondo o icono                    |
| `PronunciationSuggestionModal.jsx` | Permite al usuario sugerir una pronunciación alternativa mediante un modal |
